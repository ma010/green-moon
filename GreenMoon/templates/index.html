Features

Built by LAMP (Linux+Apache+SQL+Python).

Front End

Besides basic HTML5 , CSS and JavaScript, we tweak our website with Bootstrap , Font-awesome and jQuery for a more attractive modern look.

Web server

This website is hosted by Apache on a Linux server, and powered by Flask web framework and various Flask extensions.

Database

Both relational (PostgreSQL) and non-relational (MongoDB) databases are used. An abstract layer --SQLAlchemy is used to access the database conveniently.
